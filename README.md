# My Code

Personal repo for LeetCode solutions, experiments, and random coding stuff.

## Structure

```
my-code/
├── leetcode/
│   ├── easy/
│   ├── medium/
│   └── hard/
├── experiments/      # random scripts and ideas
├── snippets/         # reusable utility functions
├── learning/         # tutorials, courses, practice
├── projects/	      # functional apps
└── codewars/
```

## LeetCode File Naming

Files follow the format: `{problem_number}_{problem_name}.{ext}`

Example: `0001_two_sum.py`

Each file includes a header comment:
```python
# Problem: https://leetcode.com/problems/two-sum/
# Approach: Hash map for O(n) lookup
# Time: O(n) | Space: O(n)
```
